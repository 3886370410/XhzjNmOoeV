# 项目简介

## 核心功能
- 管理员角色功能模块
  - 学院管理：添加、修改、查询、展示、删除学院信息（需判断是否有学生未结业）
  - 专业管理：添加、修改、查询、展示、删除专业信息（需判断是否有学生未结业）
  - 课程管理：添加、修改、查询、展示、删除课程信息
  - 班级管理：添加、修改、查询、展示、删除班级信息（需判断是否有学生未结业），安排教师授课任务
  - 学生管理：添加、修改、查询、展示、删除学生信息，查看学生选课记录
  - 教师管理：添加、修改、查询、展示、删除教师信息，安排教师授课任务
  - 选课管理：查询、筛选选课记录，查看选课详情
- 教师角色功能模块
  - 个人信息管理：修改个人信息，修改登录密码
  - 教学安排：查看个人授课安排
  - 学生管理：查看授课学生信息（支持按班级、姓名筛选）
- 学生角色功能模块
  - 个人信息管理：修改个人信息，修改登录密码
  - 选课功能：查看可选课程列表，选择课程（选课）
  - 选课记录：查看已选课程（选课记录）

## 适用场景
- 教育机构内部课程管理
- 学生选课及教师授课安排
- 管理员对学院、专业、课程、班级、学生、教师等信息的维护与管理

## 技术特点
- 使用Python编程语言，结合Django框架开发
- 数据库采用MySQL
- 前端采用HTML技术
- 提供完整的项目文档，方便用户理解和维护
